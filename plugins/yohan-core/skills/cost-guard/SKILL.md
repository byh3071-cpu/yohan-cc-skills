# cost-guard — 토큰/비용 절약

## 규칙
- **부분 읽기**: 큰 파일은 필요한 구간만. 전체 덤프 금지.
- **표적 검색**: Grep/Glob로 좁힌 뒤 읽기. 무차별 나열 금지.
- **모델 매칭**: 탐색=haiku, 구현=sonnet, 판단·검증=opus, 진짜 어려운 추론만 상위/Fable. (지휘=세션 모델)
- **출력 절제**: 불필요한 전체 재출력·중복 요약 금지.
- **병렬 신중**: 의존 없는 읽기만 병렬. 추측성 대량 호출 금지.
- **에이전트 팀**: 토큰 약 7배. 정말 필요할 때만.
